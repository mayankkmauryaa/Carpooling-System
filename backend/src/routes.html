🚗 Carpooling System API

All available routes for the backend API:

🔐 Authentication

POST /api/v1/auth/register - Register new user

POST /api/v1/auth/login - Login user

GET /api/v1/auth/me - Get current user

POST /api/v1/auth/refresh - Refresh token

POST /api/v1/auth/logout - Logout user

GET /api/v1/auth/google - Google OAuth (redirect)

GET /api/v1/auth/google/callback - Google OAuth callback

POST /api/v1/auth/google/mobile - Mobile Google Sign-In

POST /api/v1/auth/google/link - Link Google account

👤 Users

GET /api/users/profile - Get profile

PUT /api/users/profile - Update profile

PUT /api/users/password - Change password

GET /api/users ?page=1&limit=20&role=driver&search=name&isActive=true

GET /api/users/drivers ?page=1&limit=20&isActive=true

GET /api/users/riders ?page=1&limit=20

GET /api/users/:id - Get user by ID

PUT /api/users/:id/status - Toggle user status (admin)

DELETE /api/users/:id - Delete user (admin)

GET /api/users/:userId/reviews - Get user reviews

🚗 Vehicles

GET /api/vehicles - Get my vehicles

POST /api/vehicles - Add vehicle (driver)

GET /api/vehicles/all ?page=1&limit=20&isActive=true

GET /api/vehicles/:id - Get vehicle

PUT /api/vehicles/:id - Update vehicle

DELETE /api/vehicles/:id - Delete vehicle

PUT /api/vehicles/:id/status - Toggle status

GET /api/vehicles/driver/:driverId - Get by driver

🚙 Rides

GET /api/rides - Get my rides

POST /api/rides - Create ride (driver)

GET /api/rides/search ?pickupLat=37.7749&pickupLng=-122.4194&dropLat=37.4028&dropLng=-122.0869&radius=15&availableSeats=2

GET /api/rides/recommendations - Get recommendations

GET /api/rides/all ?page=1&limit=20&status=active

GET /api/rides/:id - Get ride details

PUT /api/rides/:id - Update ride

DELETE /api/rides/:id - Cancel ride

GET /api/rides/:id/requests - Get requests

PUT /api/rides/:id/requests/:riderId ?action=approve|reject

POST /api/rides/:id/join - Request to join

GET /api/rides/my-requests - My requests

DELETE /api/rides/:id/join - Cancel join request

GET /api/rides/date/:date ?page=1

GET /api/rides/upcoming - Upcoming rides

GET /api/rides/nearby ?lat=37.7749&lng=-122.4194&radius=10

📍 Trips

GET /api/trips - Get my trips

GET /api/trips/all ?page=1&limit=20&status=completed

GET /api/trips/:id - Get trip details

POST /api/trips/:id/start - Start trip (driver)

POST /api/trips/:id/complete - Complete trip (driver)

POST /api/trips/:id/cancel - Cancel trip

GET /api/trips/driver/:driverId - Trips by driver

GET /api/trips/rider/:riderId - Trips by rider

GET /api/trips/ridepool/:ridePoolId - Trip by ride pool

GET /api/trips/date/:date - Trips by date

GET /api/trips/status/:status - Trips by status

GET /api/trips/upcoming - Upcoming trips

GET /api/trips/stats - Trip statistics

🔒 Privacy

POST /api/privacy/call/initiate - Initiate masked call

POST /api/privacy/call/end - End call

POST /api/privacy/message/send - Send in-app message

GET /api/privacy/message/conversation/:userId

GET /api/privacy/masked-phone/:userId - Get masked phone

POST /api/privacy/sos/alert - Send SOS alert

GET /api/privacy/sos/history - SOS history

GET /api/privacy/settings - Get privacy settings

PUT /api/privacy/settings - Update settings

GET /api/privacy/profile-visibility - Get visibility

PUT /api/privacy/profile-visibility - Update visibility

⭐ Reviews

POST /api/reviews - Create review

GET /api/reviews/user/:userId ?page=1&limit=20

GET /api/reviews/trip/:tripId - Trip reviews

GET /api/reviews/all ?page=1&limit=20&type=driver-to-rider

GET /api/reviews/my-reviews - My reviews

GET /api/reviews/:id - Get review

DELETE /api/reviews/:id - Delete review

GET /api/reviews/stats/user/:userId - Review stats

💬 Messages

GET /api/messages ?userId=xxx&page=1

POST /api/messages - Send message

GET /api/messages/conversations - All conversations

GET /api/messages/unread-count - Unread count

GET /api/messages/conversation/:userId ?page=1

PUT /api/messages/read - Mark as read

PUT /api/messages/read/:userId - Mark conversation read

DELETE /api/messages/:messageId - Delete message

DELETE /api/messages/conversation/:userId - Delete conversation

🌐 System

GET /api/health - Health check

GET /api/stats - System statistics

Note: Most endpoints require authentication. Add header: Authorization: Bearer <token>
